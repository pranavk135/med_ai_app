# MedAI – AI Powered Medical Assistant with Prescription Scan & Emergency Routing

MedAI is an AI-powered healthcare assistant designed to provide quick symptom analysis, prescription understanding, and emergency guidance using modern AI models.
The project was built as an MVP for hackathon and innovation submissions, focusing on real-world usability, safety, and accessibility.
 
Disclaimer: This project is for educational and research purposes only. It does not provide real medical diagnosis and should not replace professional healthcare advice.

##  Problem

Access to quick medical guidance is difficult in emergency situations, especially when users do not understand prescriptions, symptoms, or severity of their condition.

Many people:

* Cannot interpret prescriptions
* Do not know when a symptom is serious
* Do not know when to seek emergency help
* Need fast preliminary guidance before consulting a doctor

This project aims to provide a safe AI-based assistant to help users understand their condition and guide them to the next step.

---

##  Solution

MedAI provides an AI-powered interface that allows users to:

* Enter symptoms and get AI-generated insights
* Upload prescription PDFs for analysis
* Detect possible emergency conditions
* Route users to emergency guidance page
* Provide safe disclaimers and recommendations

The system focuses on usability, speed, and clarity while keeping medical safety in mind.

##  How It Works

1. User opens landing page
2. User enters symptoms or uploads prescription
3. AI model analyzes input
4. System generates insights
5. Emergency logic checks severity
6. Emergency page shown if needed
7. User receives guidance

Flow:

User → UI → API → AI Model → Result
↘ OCR / Scan
↘ Emergency Logic
